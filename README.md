# Calculadora Santander

Calculadora web com operações básicas (soma, subtração, multiplicação, divisão, exponenciação) integrada com Firebase para salvar dados de clientes.

## 🚀 Como Usar

1. Clone:
```bash
git clone https://github.com/rlehmannsilva35-stack/calculadora.git
```

2. Abra `calculadora.html` no navegador

3. Preencha: Nome, Idade, Número 1 e Número 2

4. Clique "Calcular" e veja os resultados

## 🛠️ Tecnologias

- HTML5, CSS3, JavaScript
- Firebase (armazenamento de clientes)
